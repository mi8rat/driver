# 📝 Lightweight C CMS with SQLite

A simple **Content Management System (CMS)** built in C using SQLite for persistent storage.  
Supports basic CRUD operations (Create, Read, Update, Delete) for blog posts via a CLI interface.

---

## 🚀 Features

- Create posts  
- List all posts  
- View a single post  
- Update posts  
- Delete posts  
- Persistent storage using SQLite  

---

## 🛠️ Tech Stack

- **Language:** C  
- **Database:** SQLite
